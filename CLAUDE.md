# Проект: сайт Latitudo на 1С‑Битрикс

## Что это
Корпоративный сайт компании **Latitudo** (террасная доска, заборы, фасады из ДПК).
Делается на CMS **«1С‑Битрикс: Управление сайтом»** (PHP). Дизайн готов — **исходные макеты живут
в Figma** и достаются через API по токену.

## Суть сайта (из ТЗ)
- Домен `latitudo.pro`. **5 филиалов = 5 поддоменов:** `msk` (Москва), `belgorod`, `vrn` (Воронеж),
  `krd` (Краснодар), `rnd` (Ростов). Сайт один, контент части блоков меняется по поддомену (городу).
- **1 главная страница + 6 страниц‑разделов** (Террасная доска, Строительство террас, Заборы, Перила и
  ограждения, Ступени, Все изделия из ДПК). URL раздела: `msk.latitudo.pro/stupeni`.
- Пункты меню = **якоря** на блоки. Детальных страниц (товаров, портфолио, отзывов, акций) **нет**.
- **Инфоблоки** (контент, редактируемый в админке): «Каталог продукции» (6 разделов), «Реализованные
  проекты», «Отзывы», «Магазины / Регионы» (5 городов — телефоны/адреса/реквизиты по поддоменам).
- **Включаемые области** (сквозные блоки для всех страниц): Преимущества, О компании, Дилерам и партнёрам,
  «Посетите магазин в …», Контакты, шапка, подвал.

### Рецепт обращения к API (Windows / Git Bash + Python)
Токен лежит в файле `figma.token` в корне проекта (закрыт от HTTP через `.htaccess`,
**не деплоить на прод**). Читать: `TOKEN=$(tr -d ' \r\n' < figma.token)`.
Если Figma вернёт `Invalid token` — токен протух, пересоздать в Figma и заменить файл.
```bash
TOKEN=$(tr -d ' \r\n' < figma.token); KEY='TdsV8RndMihig9YUyEJ5CF'
# 1) Структура файла (страницы и фреймы):
curl -s -H "X-Figma-Token: $TOKEN" "https://api.figma.com/v1/files/$KEY?depth=2"
# 2) Конкретные узлы (геометрия, цвета, текст):
curl -s -H "X-Figma-Token: $TOKEN" "https://api.figma.com/v1/files/$KEY/nodes?ids=389:11470&depth=2"
# 3) Рендер узла в картинку → вернёт S3-URL, его и качать:
curl -s -H "X-Figma-Token: $TOKEN" "https://api.figma.com/v1/images/$KEY?ids=389:14156&format=png&scale=2"
#    Для иконок/логотипа: format=svg
```
Грабли окружения:
- Парсить JSON и печатать кириллицу — Python с `PYTHONUTF8=1` (иначе `UnicodeEncodeError` в cp1252).
- Windows-Python **не видит** `/tmp` как Git Bash: класть временные файлы внутрь проекта.
- Убирать `\r` из URL-списков перед скачиванием (`tr -d '\r'`), иначе ломается путь.
